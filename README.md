# ChatGPT-Ollama-Repo
Welcome To ChatGPT Ollama Repo!
Today We Are Showing How To Setup This Bad Boy On Ollama!
# Setup
Type: ``git clone https://github.com/creuawards-max/ChatGPT-Ollama-Repo.git``
Then, Type: ``cd ChatGPT-Ollama-Repo``
And Type: ``./setup``
# Installation
This Installation Requires 4.9GB Avaliable On Your Phone (Or PC) So Remove Unrequired Files Or Photos
I Preffer More Executing The Command: ``./setup-mini`` That Uses 637MB Avaliable On Your Phone (Or PC)
# Finishing
If Installation Is Sucsess, Run: ``ollama create chatgpt:llama3.1 -f Dockerfile``, Mini Version: ``ollama create chatgpt:tinyllama -f Dockerfile``
